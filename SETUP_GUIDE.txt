================================

2. Database file (chatbot.db) is created
3. Open http://localhost:5000 in browser
4. Click "New Chat" to create session
5. Type message and send
6. Message saved to database
7. Sent to OpenRouter API
8. LLaMA 3 processes with context
9. Response returned and displayed
10. Both messages saved to database

🐛 QUICK TROUBLESHOOTING
========================

Port already in use?
   → Change port in app.py line 99 to 5001

API key expired?
   → Update OPENROUTER_API_KEY in app.py line 16

Dependencies not found?
   → pip install -r requirements.txt

Database locked error?
   → Delete chatbot.db and restart
